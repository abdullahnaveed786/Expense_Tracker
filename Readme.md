# 💰 Personal Expense Tracker

A command-line application to track personal expenses,
built with Python and OOP principles.

## Features
- Add expenses with description, amount, and category
- Persistent storage via CSV
- View spending by category with overspend warnings
- Monthly summary
- Balance tracking with debt detection

## How to Run
git clone https://github.com/AbdullahNaveeed/Expense_Tracker
cd Expense_Tracker
python main.py

## Tech Stack
- Python 3
- CSV module
- datetime module
- Object-Oriented Design
